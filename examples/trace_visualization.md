# Reasoning Trace Visualization

This guide explains how to interpret the JSONL traces generated by Q-Verify.

Each line in `data/raw_traces/` contains:

```json
{
  "id": 0,
  "task": "GHZ_State",
  "hallucination_rate": 0.1,
  "reward": -0.5,
  "ftle_stability": -0.12,
  "trajectory_length": 15
}
